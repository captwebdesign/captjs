How to use this demo

  1. Run npm install and grunt to make dist/capt.js available.
  2. Open your brower's console and try out the various commands.

Available Functions:

  • capt.cancelAlarm(alarm_id)

    Use to cancel an alarm previously set using capt.setAlarm().
    @param alarm_id Int Is the return value from capt.setAlarm().

  • capt.currentTimestamp()

    Returns the current date as milliseconds since 1970 00:00:00 UTC.

  • capt.dateFromTimestamp(timestamp)

    Use this function to return a date string from a timestamp such as that from capt.currentTimestamp().
    @param timestamp Int|String Is a date compatible timestamp.

  • capt.helloWorld()

    Use to output the string "Hello World!" to the browser's console.

  • capt.isBrowserLoaded()

    Returns true if the browser has loaded or false if it has not.

  • capt.setAlarm(delayInMilliseconds, callback)

    Use to set an alarm with a callback.
    @param delayInMilliseconds Int Is the total milliseconds to delay before calling the callback.
